Política de Privacidade - Smart Unit Converter

Última atualização: 21 de junho de 2025

1. Informações Gerais

O Smart Unit Converter é um aplicativo de conversão de unidades de medida que funciona completamente offline. Esta política descreve como tratamos suas informações ao usar nosso aplicativo.

2. Informações que Coletamos

2.1 Informações Pessoais

Não coletamos informações pessoais identificáveis como nome, email, telefone ou endereço.

2.2 Dados de Uso

  • Preferências do usuário: Salvamos suas configurações e preferências localmente no dispositivo usando SharedPreferences (unidades favoritas, configurações de idioma, etc.)
  • Histórico de conversões: Mantemos um histórico local das conversões realizadas para facilitar o uso

2.3 Dados Técnicos

Não coletamos dados técnicos como localização, informações do dispositivo ou dados de crash

3. Como Usamos as Informações

As informações armazenadas são usadas exclusivamente para:

  • Melhorar sua experiência de uso do aplicativo
  • Manter suas preferências entre sessões
  • Fornecer acesso rápido ao histórico de conversões

4. Compartilhamento de Informações

Não compartilhamos, vendemos ou transferimos suas informações para terceiros. Todas as informações permanecem armazenadas localmente em seu dispositivo.

5. Armazenamento de Dados

  • Todos os dados são armazenados localmente no seu dispositivo
  • Não utilizamos servidores externos ou serviços de nuvem
  • Os dados são mantidos apenas enquanto o aplicativo estiver instalado

6. Segurança

Implementamos medidas de segurança apropriadas para proteger suas informações:

  • Armazenamento local seguro através das APIs nativas do Android
  • Não transmitimos dados pela internet
  • Não requeremos permissões desnecessárias

7. Seus Direitos

Você tem o direito de:

  • Acessar: Visualizar todas as suas configurações através do aplicativo
  • Deletar: Limpar o histórico e configurações através das opções do app
  • Controlar: Modificar suas preferências a qualquer momento

8. Retenção de Dados

  • Os dados são mantidos apenas enquanto o aplicativo estiver instalado
  • Ao desinstalar o aplicativo, todos os dados são automaticamente removidos
  • Você pode limpar os dados manualmente através das configurações do app

9. Alterações nesta Política

Podemos atualizar esta política ocasionalmente. Quando isso acontecer:

  • A data de "Última atualização" será modificada
  • Mudanças significativas serão comunicadas através de atualizações do app

11. Conformidade

Esta política está em conformidade com:

  • Lei Geral de Proteção de Dados (LGPD) - Brasil
  • General Data Protection Regulation (GDPR) - União Europeia
  • California Consumer Privacy Act (CCPA) - Estados Unidos

Resumo

O Smart Unit Converter é um aplicativo que respeita sua privacidade. Não coletamos dados pessoais, não usamos internet para transmitir informações e tudo funciona offline no seu dispositivo.
